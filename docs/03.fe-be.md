# 프론트엔드? 백엔드?

---

## 프론트엔드

- 보통은 클라이언트에서
- 서버가 가져온 데이터를 화면에 보여주는 역할
- 언어: HTML, CSS, JS

## 백엔드

- 보통은 서버에서
- 프론트엔드에게 데이터를 제공하는 역할
- 언어: 자유, 보통은 Java, Python, Ruby, PHP, Node.js, Go
- DB, 서버, 네트워크 등의 기술을 사용
